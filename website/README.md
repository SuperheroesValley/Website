simple theme for hugo

# Live Demo
https://gangjun06.github.io/SimpleIntroDemo


# Get Started

```bash
hugo new site <site-name>
cd <site-name>
git submodule add https://github.com/gangjun06/SimpleIntro themes/SimpleIntro
```
